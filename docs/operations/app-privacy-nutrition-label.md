# App Privacy / Nutrition Label Notes

## Data Linked to the User
- user identifiers for account continuity
- purchase and subscription state
- product interaction events
- coarse diagnostics and crash breadcrumbs

## Data Not Used for Tracking
- launch scope does not include third-party ad tracking
- no public social profile data is shared for ad targeting

## Sensitive Data Notes
- HealthKit access is optional and limited to authorized domains
- raw HealthKit payloads are not exported into analytics

## Purpose Categories
- app functionality
- analytics
- developer diagnostics

## External Gap
Final App Store Connect questionnaire entry still requires submission-time verification against configured SDKs and build settings.
