# Sub-Agent Context Snippet

Paste or attach this context to any sub-agent task before the task-specific brief.

Aevora is a native iPhone life-RPG where consistent real-life habits create visible in-world progress. The product exists because most habit trackers are functional but emotionally flat; Aevora makes progress feel witnessed, meaningful, and motivating.

v1 is not a broad life OS or an MMO. It is a utility-first, single-player-first habit product with a lightweight explorable fantasy layer. The user chooses a fantasy identity, completes daily vows quickly, and sees their avatar and the launch district of Cyrane/Dawnmarch visibly progress through a 7-day starter arc and a 30-day Chapter One.

Hard rules:
- logging a vow must always be faster than visiting the world
- build for portrait iPhone first
- lore is flavor, not friction
- manual logging is central; verified logging is additive
- do not invent scope beyond locked v1
- do not create a second source of truth
