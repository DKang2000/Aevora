# NC First Playable Core Loop Pack

## Purpose
Author the launch narrative layer needed for the first playable loop and the 7-day starter arc without fragmenting the shared mechanics.

## Inputs / dependencies
- `AGENTS.md`
- `docs/product/05_Narrative_Content_Launch_Spec.md`
- `docs/specs/source-of-truth/ST-06_content_config_schema.md`
- `docs/specs/source-of-truth/ST-10_string_key_registry_copy_token_structure.md`

## Sections
### NC-02 — Origin family overview pack
- Dawnbound: civic defense, warding, shared courage.
- Archivist: pattern, memory, study, repair through understanding.
- Hearthkeeper: nourishment, warmth, care, steadiness.
- Chartermaker: stewardship, logistics, contracts, civic order.

### NC-03 — Identity intro pack for 10 identities
- Dawnbound: Knight, Paladin, Ranger.
- Archivist: Mage, Scholar.
- Hearthkeeper: Farmer, Baker, Cafe Keeper.
- Chartermaker: Merchant, Charterlord.
- Identity rule: flavor overlays change framing and dialogue, never the core progression engine.

### NC-04 — First session script
- Opening frame: the user arrives in a dimmed Ember Quay during a fragile Rekindling.
- Maerin Vale introduces the district’s broken cadence.
- The user chooses who they are, accepts a starter path, and proves that steady action still matters.

### NC-05 — First magical moment variants
- Primary: the shuttered oven glows alive for a breath.
- Additional launch-safe variants: ward lantern relit, canal gate stirs, Pollen appears, district sigil brightens.
- All variants resolve to the same systemic outcome: day-one world response in Ember Quay.

### NC-06 — 7-day starter arc full script
- Arc title: `The Ember That Returned`.
- Day beats: arrival, ash in the stones, missing craft clue, canal rumors, cooling night, quarter gathers, oven rekindled.
- The arc ends with the bakery reopening and a clear invitation into tomorrow and Chapter One.

### NC-08 — NPC cast bible for launch NPCs
- Maerin Vale: practical guide and witness.
- Sera Quill: lore and journal guide.
- Tovan Hearth: emotional anchor of the bakery arc.
- Brigant Hal: discipline and effort framing.
- Ilya Fen: civic order and charter logic.
- Pollen: silent wonder and attachment companion.

### NC-09 — NPC dialogue pack
- Dialogue sets are day-aware and keyed by quest beat.
- Each family gets light framing overlays rather than fully bespoke branches.
- Dialogue always clarifies the practical action before the mythic flourish.

### NC-10 — Quest journal text pack
- Each starter-arc day has a plain-language quest title, a one-line summary, and a tomorrow-facing continuation.
- Journal copy prioritizes “what to do next” over lore density.

### NC-11 — Reward / level-up / chapter-complete copy pack
- Reward copy reinforces that the city responds to kept cadence.
- Level-up copy celebrates continuation, not dominance.
- Day-7 completion copy closes the bakery arc and points toward the next return.

### NC-14 — Diegetic lexicon / allowed lore vocabulary
- Allowed core flavor terms: Resonance, Chains, Embers, Rekindling, Hearth, Witness, cadence.
- Contracts and utility UI still use plain system nouns first where ambiguity matters.
- Disallowed tone: chosen-one language, shame-heavy failure framing, juvenile parody fantasy.

## Output / canonical artifact
- This narrative pack
- Expanded content payload and launch copy keys in `content/launch/`

## Edge cases
- Identity variants must remain additive flavor, not alternate system trees.
- Tomorrow prompts stay encouraging when a user is in cooling state.
- Reward narration cannot obscure the utility facts of what changed.

## Explicit non-goals
- full Chapter One script production
- broad art-production direction beyond placeholder-clean copy hooks
