# UX Durable Starter Arc Retention Pack

## Purpose
Lock the utility-first return surfaces that make the 7-day starter arc feel continuous instead of day-one-only.

## Inputs / dependencies
- `AGENTS.md`
- `docs/specs/ux/UX-first-playable-core-loop-pack.md`
- `docs/specs/source-of-truth/ST-09_error_empty_offline_state_catalog.md`

## Sections
### UX-10 / UX-11
- Vow cards keep quick logging primary and expose recent progress context inline.
- History is lightweight and glanceable; logging still stays faster than navigation.

### UX-13
- Today owns the current day objective.
- Quest Journal makes completed/current/upcoming beats readable at a glance.

### UX-17 / UX-18
- Hearth is the emotional destination for earned starter-arc props.
- Inventory remains intentionally small and reward-led in this bundle.

### UX-22 / UX-25
- Return prompts appear after value and reinforce tomorrow-facing continuity.
- Cooling is framed as recoverable and non-shaming.

## Output / canonical artifact
- This pack
- Supporting copy and surfaces in Today, Quest Journal, World, and Hearth

## Edge cases
- Missing content should degrade safely to utility-first copy.
- Return prompts must not obscure the next concrete action.

## Explicit non-goals
- Full notification settings stack
- Lore-heavy journal presentation
