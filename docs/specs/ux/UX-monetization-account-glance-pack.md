# UX Monetization, Account, and Glance Pack

## Purpose
Lock the user-facing behavior for the soft paywall, account continuity, and return surfaces in the first monetizable beta slice.

## Inputs / dependencies
- `AGENTS.md`
- `docs/specs/source-of-truth/ST-07_subscription_entitlement_matrix.md`
- `docs/specs/source-of-truth/ST-08_permission_matrix.md`

## Sections
### UX-08 / UX-20 / UX-21
- The paywall appears only after the first magical moment and may be dismissed into the free path.
- Trial, purchase, restore, and downgrade language reinforce continuity rather than threat.
- Account/settings surfaces make entitlement state legible without burying logging speed.

### UX-22 / UX-23 / UX-24
- Notification education happens after value and explains why return prompts matter.
- One free widget reflects Today/starter-arc utility first.
- Premium glance breadth deepens witness context but never removes the base utility loop.

## Output / canonical artifact
- This pack
- Supporting copy, remote-config defaults, widget labels, and profile/settings surfaces in the repo

## Edge cases
- Restore states must degrade clearly when nothing is available to restore.
- Notification denial must not dead-end the user.
- Premium gating must never block the free 7-day starter arc.

## Explicit non-goals

- full chapter-one monetization expansion
