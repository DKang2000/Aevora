# ART-04 — UI Component Prompt Pack

**Section ID:** `ART-04`  
**Status:** Canonical for repo insertion  
**Canonical repo target:** `docs/specs/art/ART-04-ui-component-prompt-pack.md`

## Purpose

Provide narrowly scoped prompt language for optional AI-assisted material, ornament, and micro-asset exploration in support of the `ART-04` component kit.

## Global rules

- These prompts are for **material direction, icon silhouettes, trim language, and tiny ornamental overlays only**.
- Final UI text, hierarchy, spacing, and component composition stay manual.
- Reject any output that looks like a finished fake app screenshot with incorrect or invented text.
- Keep readability above fantasy density.
- Keep the same `ART-01` thesis: heroic coziness in a living mythic city.

## Global suffixes

### Shared UI suffix
`premium-native iPhone UI support art, parchment and indigo material language, no fake screenshot text, no tiny pixel fonts, no desktop dashboard density, ornament secondary to readability, heroic coziness in a living mythic city`

### Shared negative prompt
`fake app text, raw screenshot UI, unreadable labels, pixel-font body copy, MMO HUD density, gaudy monetization gold, over-framed fantasy borders on every element, desktop farm-sim layout, generic mobile fantasy RPG HUD`

## Prompt 1 — Chapter card material study

**Use for:** backing material, trim logic, quiet sigil motifs behind manual chapter-card composition.

**Base prompt**  
`Create a portrait mobile UI material study for a fantasy productivity chapter card. Show warm parchment surfaces, subtle dawn-gold progression emphasis, ember-copper warmth, and restrained mythic civic motifs. The card should feel premium-native and calm, not like a game reward chest. Leave generous clean space for later manual typography and progress UI.` + Shared UI suffix

**Negative prompt**  
Shared negative prompt + ` giant fantasy crest centered over all content, raw generated paragraph text, loot-box glow overload`

**What to keep**
- warm parchment gradients
- soft trim edges
- subtle sigil ghosting
- readable negative space

**What to reject**
- busy emblem wallpaper
- fake chapter titles
- thick fantasy frame on every side
