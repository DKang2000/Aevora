# ART-06 — Avatar Prompt Pack

**Section ID:** `ART-06`  
**Use:** optional later external exploration only. The deterministic repo renderer remains the canonical execution path unless a bake-off is explicitly requested.

## Global style anchor
Aevora avatar base system, portrait-first iPhone game companion, premium-native UI framing, heroic coziness in a living mythic city, warm mythic-civic materials, readable launch avatar silhouettes, civilian and scholarly dignity equal to martial expression, larger-than-retro readability, simple shapes over micro-detail, clean touch-safe presentation.

## Global negatives
Do not generate: MMO armor spam, giant weapons, tiny unreadable faces, full pixel font labels, desktop RPG equipment panes, over-dark grim fantasy, glossy mobile-gacha rendering, chibi proportions, hypersexualized bodies, tiny jewelry clutter, ultra-detailed fabric noise, sideways landscape compositions.

## Prompt 1 — base frame silhouette board
Create a portrait target sheet showing four launch-ready avatar base silhouettes for Aevora. The sheet should feel like a production-facing avatar base system board, not character splash art. Show neutral standing poses, one bust crop per silhouette, clear silhouette IDs, and a calm premium-native presentation. The world mood is warm mythic civic fantasy: parchment neutrals, ember warmth, indigo depth, moss restoration. The silhouettes should read instantly on an iPhone. Civilian, scholarly, mercantile, and quietly martial notes should all feel valid without any one dominating.

## Prompt 2 — head / hair / skin anchor board
Create a portrait target sheet for Aevora avatar head, hair, and skin anchors. Include a starter skin-tone matrix, readable launch hair silhouettes, accessory anchor guidance, and one small portrait-size proof that the face area remains readable at phone scale. Keep the art premium-native and system-oriented, not painterly splash art. Emphasize warmth, dignity, and readability.

## Prompt 3 — palette / material application board
Create a portrait target sheet showing how Aevora avatar palettes apply across cloth, trim, and neutral materials. Repeat one avatar base silhouette across multiple palette treatments. Show palette swatches and small material callouts. Keep colors intentional, warm, and mythic-civic. The goal is not many colors; it is clear, reusable application logic.

## Prompt 4 — onboarding avatar preview board
Create a portrait target sheet for the onboarding avatar selection surface in Aevora. Show one fast, touch-friendly preview card with silhouette and palette selector affordances and optional accessory row treatment. This must feel like one fast setup step, not a long character creator. Keep the UI premium-native and clean.

## Prompt 5 — Hearth avatar preview board
Create a portrait target sheet for the Hearth avatar presentation in Aevora. Show how the selected avatar appears alongside home/reward cues in a warm restorative surface. Include one low-collection variant and one more-settled variant. The composition should prove that Hearth can be avatar + home/customization, not inventory-only.
