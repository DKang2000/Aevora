# ART-05 — Onboarding Prompt Pack

**Section ID:** `ART-05`  
**Status:** Canonical for repo insertion  
**Canonical repo target:** `docs/specs/art/ART-05-onboarding-prompt-pack.md`

## Purpose

Provide narrowly scoped prompt language for optional later external exploration in support of the onboarding board set.

## Global rules

- These prompts are for scene fragments, ornamental accents, silhouette guidance, and emotional stills only.
- Do not trust model-generated UI text, labels, or layouts.
- Final boards must remain manually composed in repo-native exports.
- Keep the Aevora thesis: heroic coziness in a living mythic city.
- Keep onboarding utility-first. Art should support trust and belief, not decorative noise.

## Prompt 1 — Welcome promise backdrop fragment

Create a portrait-friendly onboarding backdrop fragment for a premium-native iPhone life-RPG. Warm parchment dawn light, indigo depth at the edges, subtle civic-fantasy materials, gentle lantern glow, no full environment scene, no characters, no UI text, no HUD, no fake mobile screenshot. The feeling is hopeful, calm, and inviting rather than bombastic.

Negative prompts:
- fake app screenshot
- text-heavy poster
- medieval parchment scroll
- dense RPG menu
- landscape composition
- neon fantasy UI

## Prompt 2 — Problem / solution carousel icon-language strip

Create a clean set of three small visual motifs for a premium iPhone onboarding carousel: dead tracker fatigue, humane recovery after a miss, and visible progress warming a district. Minimal icon + prop language, soft edges, warm parchment neutrals, ember gold accents, indigo anchors, readable at small size, no text, no big scenes.

Negative prompts:
- infographic with labels
- skeuomorphic dashboard
- grim dark fantasy
- maximal detail
- noisy symbols

## Prompt 3 — Origin family card accents

Create four selection-card accent fragments for a mythic civic-fantasy onboarding flow: Dawnbound, Archivist, Hearthkeeper, Chartermaker. Each accent should imply a different material + prop language: civic warmth, archives and sigils, domestic craft, contracts and trade. Touch-friendly, premium-native, no full character portraits, no text, no ornate borders.

Negative prompts:
- trading-card frame
- MMO class select
- giant weapons
- comic-book action pose
- overdecorated fantasy crest

## Prompt 4 — Identity silhouette support strip

Create a portrait-safe strip of identity silhouette fragments for launch identities in a warm fantasy city. Distinct reads through posture, outer shape, and props; readable on mobile; civilian, scholarly, mercantile, and martial identities all matter. No facial-detail dependency, no names, no card text, no full-body battle lineup.

Negative prompts:
- identical anime faces
- armor-heavy class grid
- overrendered painterly portraits
- tiny unreadable accessories
- dark gritty palette

## Prompt 5 — First magical moment still

Create one emotional still for the first magical moment in a cozy mythic canal district: a faintly dead civic object rekindles because the player kept one small vow. Options include a canal lantern relighting, a warded oven glowing, or a dim quay marker brightening. Portrait composition, visible warmth returning, premium game-adjacent but still iPhone-safe, not a combat scene, not cinematic wide-screen.

Negative prompts:
- explosion
- battle
- giant monster
- cinematic letterboxing
- cluttered interface
- text overlay

## Prompt 6 — Soft paywall premium halo fragment

Create a soft premium-upgrade accent fragment for a subscription sheet that follows a humane first magical moment. Invitation, warmth, possibility, and chapter depth, not pressure. Use warm gold halo cues, refined civic-fantasy materials, and a subtle sense of expanded world depth. No pricing text, no buttons, no mock checkout UI, no aggressive conversion energy.

Negative prompts:
- casino glow
- neon CTA
- sales-banner typography
- heavy lock icons
- manipulative urgency
- faux screenshot
